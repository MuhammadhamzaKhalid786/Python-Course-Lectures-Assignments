## Description
This is a **console-based Python Smart Assistant** built using only core Python concepts.  
It can greet the user, answer simple questions, and exit when the user types 'bye' or 'exit'.

## Concepts Used
- print() and input()
- Variables and Data Types
- Loops (while loop)
- Conditional Statements (if, elif, else)
- Functionss
- Classes and Objects
- Modules (import from another file)
- Lists and Dictionaries (to store Q&A)
- String Matching
